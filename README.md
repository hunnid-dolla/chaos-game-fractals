# Fractal Flame Generator
Генератор изображений фрактального пламени (Fractal Flame) на основе алгоритма Chaos Game. Программа визуализирует математическую красоту систем итерируемых функций (IFS), используя современные методы рендеринга и оптимизации производительности.

## Ключевые фичи

*   **Математическое ядро:**
    *   Реализован алгоритм **Chaos Game** с поддержкой цветной генерации.
    *   Поддерживается **5 нелинейных трансформаций**: `linear`, `sinusoidal`, `spherical`, `swirl`, `horseshoe`.
    *   Применяются аффинные преобразования (сжатие, вращение, сдвиг).

*   **Гибкая конфигурация:**
    *   Настройка через CLI аргументы или внешний JSON-файл.
    *   Валидация входных данных с помощью `pydantic`.
    *   Возможность задавать собственные аффинные коэффициенты или генерировать их случайно.

*   **Рендеринг и постобработка:**
    *   **Логарифмическая гамма-коррекция** для достижения эффекта реалистичного "свечения" на черном фоне.
    *   **Поддержка вращательной симметрии** для создания фрактальных мандал и снежинок.
    *   Сохранение результата в формате PNG (RGB, 8 бит на канал).

## Оптимизация и архитектура

Проект спроектирован с упором на максимальную производительность:
*   **Векторизация:** Вся математика рендеринга выполняется на `numpy`, что ускоряет вычисления в десятки раз по сравнению с чистым Python.
*   **Многопоточность:** Используется `ProcessPoolExecutor` для параллельной генерации фрактала на всех доступных ядрах процессора.
*   **Архитектура:** Код разделен по принципу разделения ответственности (`core`, `renderer`, `transformations`), использует `dataclasses` и строгую типизацию.

## Технологии
`Python`, `NumPy`, `Pydantic`, `Poetry`, `Ruff`, `Pytest`, `multiprocessing`.

## Запуск

1.  **Установка зависимостей:**
    ```bash
    poetry install
    ```

2.  **Запуск через CLI:**
    ```bash
    # Генерация "Галактики"
    poetry run python -m src.main -s 200000 -i 5000 -t 4 --symmetry 3 -f swirl:1.0,horseshoe:0.5 -o galaxy.png
    ```

3.  **Запуск через JSON-конфиг:**
    *   Создайте файл `config.json`:
        ```json
        {
          "size": {"width": 1920, "height": 1080},
          "samples": 300000,
          "threads": 4,
          "gamma": 3.0,
          "symmetry_level": 6,
          "functions": [
            { "name": "sinusoidal", "weight": 2.0 },
            { "name": "swirl", "weight": 0.5 }
          ]
        }
        ```
    *   Запустите генерацию:
        ```bash
        poetry run python -m src.main --config config.json --output-path neon-flower.png
        ```
